# Sentencia Switch

Conocida como seleccion multiple
A veces es necesario comparar el valor de una variable con una serie de valores
concretos.

```java

switch(variable) { // operación lógica
    case valor1: // caso 1
        sentencias
        break;
    case valor2: // caso 2
        sentencias
        break;
    // .
    // .
    // .
    // pueden contiuar los casos
    default: // sino coincide niguno
        sentencias
}
```
